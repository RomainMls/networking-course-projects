# Networking Course Projects

This repository contains two networking course projects written in Java. The first one is a socket-based Tic-tac-toe game, and the second one is a Docker-based mail and DNS network with several domains.

The repository is mainly a student project archive: it keeps the source code, reports, statements, and configuration files used during the course.

## Projects

### Project 1 - Tic-tac-toe Sockets

A client/server Tic-tac-toe application using Java sockets. The server accepts multiple clients, creates a thread for each connection, supports matchmaking between players, and also provides a bot mode.

Folder: `project1-tictactoe-sockets/`

### Project 2 - Mail, DNS and Docker Network

A Docker network simulating several domains with DNS servers, routers, mail servers, and a Thunderbird client. The mail server is implemented in Java and supports SMTP, POP3, and IMAP handlers.

Folder: `project2-mail-dns-docker-network/`

## Repository Structure

- `project1-tictactoe-sockets/`: Java socket Tic-tac-toe project, report and statement.
- `project2-mail-dns-docker-network/`: Docker network, DNS/FRR configuration and Java mail server.

## Notes

Both projects were made in an academic context. The goal was not to build production-ready network software, but to understand core networking concepts through practical implementations.
